# Model-Viewer
A simple OpenGL model viewer. 


Description:
A straight forward model viewer.  

Instructions:  

Make:  
```$ make```   
which will create the executable: "model_viewer"  

To Run:  
```./model_viewer file```  
   where:  
      *file is an .obj model file
